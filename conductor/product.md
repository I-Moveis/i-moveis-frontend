# Product Guide: Apartment Rental App

## Overview
A mobile/web application for renting apartments, built using Flutter. This frontend repository connects to a backend API to deliver a seamless apartment discovery and management experience.

## Target Audience
- **Renters / Tenants:** Individuals looking to find and rent apartments.
- **Landlords:** Property owners listing and managing their properties.
- **Admins:** System administrators who manage the platform and oversee users/listings.

## Core Focus
**Discovery & Booking:** The application prioritizes providing the best possible search, filtering, and booking experience for renters looking for their next home.

## Key Features (Initial Release)
- **Advanced Search & Media:** Comprehensive search filters, map views, and high-quality image galleries for property exploration.
- **In-app Messaging:** Direct and secure communication channels between renters and landlords.
- **Smart Notifications:** Automated, intelligent alerts for new listings, unread messages, and booking status updates.

## Look & Feel
The visual style and user experience will be dictated by external designs provided via Figma.
