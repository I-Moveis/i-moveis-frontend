# Tech Stack

## Core Language & Framework
- **Framework**: Flutter (Mobile & Web)
- **Language**: Dart

## Architecture & State Management
- **State Management**: Riverpod (Provides compile-safe, robust, and highly scalable modern state management.)

## Routing & Navigation
- **Navigation**: go_router (Declarative routing that excels at deep linking and web support.)

## Networking & Data Fetching
- **API Client**: Dio (Powerful HTTP client with support for interceptors, global configuration, and FormData.)

## Local Storage
- **Storage**: flutter_secure_storage (Encrypted key-value storage, ideal for securely storing authentication tokens.)
